## 📘 What is jax_tutorial?

jax_tutorial is a guide that helps you learn JAX, a tool used to speed up math and science code. The tutorial focuses on how JAX applies to astrophysics, which is the study of stars, planets, and space. It shows how to use JAX for tasks like faster calculation, finding exact math gradients, and running many calculations at once.

This tutorial is made for people studying astrophysics who know some Python. Python is a programming language often used in science. You do not need to be an expert programmer, but some basic Python and math helps.

---

## 💻 What You Will Learn

- When and why to choose JAX over other tools like NumPy or TensorFlow
- How to speed up your code with JAX’s just-in-time (JIT) compiler
- How to calculate exact math slopes with JAX for optimization
- How to run many calculations at once using JAX’s automatic vectorization
- How to use functional programming with JAX to write clean and efficient code

The tutorial shows examples you can follow and run yourself.

## 🧰 About the Code

The code focuses on key features of JAX:

- **JIT compilation**: This speeds up your code by compiling it before running.
- **Automatic differentiation**: This finds exact rates of change, useful for fitting models.
- **Vectorization**: This lets JAX run many calculations at the same time.
- **Functional programming**: This style makes the code easier to test and reuse.
